# jaelee.dev

Personal website of **Jaehyung Lee** — PhD student in Materials Science & Engineering at Johns Hopkins University.

Built with [Jekyll](https://jekyllrb.com/) and the [al-folio](https://github.com/alshedivat/al-folio) theme, deployed to GitHub Pages at [jaelee.dev](https://jaelee.dev).

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000>.
